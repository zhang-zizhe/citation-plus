## How to use it

**1. Clone the repo**  
   ```bash
   cd ~
   git clone https://github.com/zhang-zizhe/citation-plus.git
   cd citation-plus
   ```

**2. Set your Scholar ID**
Open `fetch_citations.sh` (*e.g. with Vim*) and replace:
```bash

```
and change it to your Google Scholar ID.
